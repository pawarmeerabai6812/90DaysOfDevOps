# basic Commands of Devops

- pwd- it show the present working directory
- ls: it show the present working directory
- cd : it used to change the directory
- clear: it used to clear screen
- whoami:it show the current user login
- date-it show the date and time
- ls -a:List all file in current directory including hidden configuration file (those sterting with a )
- sudo apt install git: download and installed the software (git)
- useradd: create a new user accout on system
- ping : Test if a remote server is reachable by sending a small data packet.
- ip addr: show the IP addresses and network interfaces of your machine
- netstat -tuln : show all listening network port on your system.
- history - list down the history of commands 
- man: it will show what is use of commands
-  systemctl start - to start the service 
- systemcetl stop- stop the service 
- df -kh to check the disk space 
- makdir - used to create new directory
